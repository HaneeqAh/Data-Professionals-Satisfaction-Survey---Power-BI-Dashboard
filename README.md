# Data-Professionals-Satisfaction-Survey---Power-BI-Dashboard
Interactive Power BI dashboard built from a Data Professionals Satisfaction Survey dataset. Data was cleaned with Power Query and analyzed using DAX measures. The report visualizes salary trends, job satisfaction, work-life balance, roles, and tool preferences with dynamic, filterable visuals for insight discovery.

Tools & Technologies Used
>Power BI Desktop
>Power Query (Data Cleaning & Transformation)
>DAX Measures
>>Data Modeling
>Interactive Visualizations

Data Preparation
>The raw dataset required several preprocessing steps before analysis:
>Removed null and inconsistent values
>Standardized job titles and country names
>Cleaned salary and experience fields
>Created calculated columns and measures using DAX
>Structured the dataset for reporting and filtering
